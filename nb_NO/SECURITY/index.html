Security Policy¶

Supported Versions¶

Only the master branch receives security updates. When an security update is made, it is released with the latest version.

Version Supported
master ✅
anything else ❌

Reporting a Vulnerability¶

If you find a security vulnerability, please report it to the developer @niccokunzmann or open an issue if you think it is a minor thread.
